# Branches 

## Create a new branch
```bash
git branch <branch-name>
```

## Switch to a branch
```bash
git checkout <branch-name>
```

## Create a new branch and switch to it
```bash
git checkout -b <branch-name>
```

## List all branches
```bash
git branch
```

## Delete a branch
```bash
git branch -d <branch-name>
```

## Delete a branch (force)
```bash
git branch -D <branch-name>
```

## Merge a branch
```bash
git merge <branch-name>
```
